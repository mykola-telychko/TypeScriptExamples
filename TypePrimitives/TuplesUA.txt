Кортежі — це спеціальний тип даних, який дозволяє зберігати фіксовану кількість елементів, де кожен елемент має свій конкретний тип. На відміну від звичайних масивів (де всі елементи мають один тип), кортежі зберігають дані з чітко визначеною структурою.

Приклад кортежу:

typescript
const user: [string, number, boolean] = ["John", 25, true];
Тут user — це кортеж з трьох елементів: string, number і boolean.

Порядок і тип елементів фіксовані.

Основні властивості кортежів:
Фіксована довжина: Кількість елементів у кортежі заздалегідь визначена.

Типізація за індексом: Кожен індекс має свій тип.

Використання: Зручно для роботи зі структурованими даними, наприклад, API-відповідями, координатами тощо.

Приклад з API-відповіддю:

typescript
const response: [string, number] = ["OK", 200];

Перший елемент — статус повідомлення (string).
Другий елемент — код статусу (number).
